Operators

// Arithmetic Operators
+   // Addition
-   // Subtraction
*   // Multiplication
/   // Division
%   // Modulus (remainder)

// Relational (Comparison) Operators
==  // Equal to
!=  // Not equal to
>   // Greater than
<   // Less than
>=  // Greater than or equal to
<=  // Less than or equal to

// Logical Operators
&&  // Logical AND
||  // Logical OR
!   // Logical NOT

// Bitwise Operators
&   // AND
|   // OR
^   // XOR
~   // Complement
<<  // Left shift
>>  // Right shift
>>> // Unsigned right shift

// Unary Operators
+   // Unary plus
-   // Unary minus
++  // Increment
--  // Decrement

// Assignment Operators
=   // Assignment
+=  // Add and assign
-=  // Subtract and assign

^=  // Bitwise XOR and assign

>>= // Right shift and assign
